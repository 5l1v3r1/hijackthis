This tool is intended for automatically collecting crash dump of HiHackThis if it is crashed.

Instruction is available under wiki pages: https://github.com/dragokas/hijackthis/wiki/HJT:-Got-error-or-crash-%3F

How to use:
1. Place HiJackThis.exe in this folder.
2. Place here Sysinternals ProcDump: https://live.sysinternals.com/procdump.exe
3. Run GetDump.bat 'as Adminisatrator'.
4. Provide us Dumps.zip file.
